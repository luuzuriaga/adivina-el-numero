# 🎮 Juego de Adivina el Número

Un juego interactivo donde debes adivinar el número secreto que la computadora está pensando. ¡Pon a prueba tu intuición y lógica!

## 🌟 Características

- 🎯 Sistema de adivinanza con retroalimentación instantánea
- 🎨 Interfaz visual atractiva y responsiva
- 🏆 Sistema de puntuación y récords
- 💡 Pistas que te guían hacia el número correcto
- 🔄 Opción para reiniciar y jugar nuevamente

## 🎯 Cómo Jugar

1. La computadora elige un número secreto dentro de un rango establecido
2. Ingresa tu intento en el campo de texto
3. Presiona el botón "Intentar" o Enter
4. Recibe pistas: "más alto" o "más bajo"
5. ¡Continúa hasta adivinar el número correcto!

## 💻 Instalación y Uso

1. **Clona el repositorio**

2. **Navega al directorio**

3. **Abre el archivo en tu navegador**
